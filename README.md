# Conway's Game of Life
My implementation of Conway's Game of Life.

If you're not familiar with Conway's Game of Life, then [read this first](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

## Setup
1. Download and install [Git](https://git-scm.com/downloads), if you don't already have it.

2. Open terminal and run the git clone command:

   ```
   $ git clone https://github.com/cramaechi/game-of-life.git
   ```
    or [download as ZIP](https://github.com/cramaechi/game-of-life/archive/master.zip).

3. Go to the project folder:

   ```
   $ cd game-of-life
   ```

4. Compile:

   ```
   $ make
   ```
   
## Usage
Run:

```
$ ./game_of_life
```
